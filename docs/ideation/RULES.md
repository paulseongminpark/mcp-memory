# 아이디에이션 세션 규칙

> 2026-03-05 추가 / 네이밍 규칙 갱신

## 파일 네이밍

```
{세션}-r{라운드}-{번호}-{주제}.md
```

예시:
- `a-r3-16-주제.md`  (A세션 Round 3, 16번)
- `b-r3-14-주제.md`  (B세션 Round 3, 14번)
- `c-r3-10-주제.md`  (C세션 Round 3, 10번)
- `d-r3-11-주제.md`  (D세션 Round 3, 11번)

번호는 세션 내 누적 순번 (라운드 넘어도 이어감).
인덱스 파일(`{세션}-index.md`)은 라운드 없이 유지, 내용만 갱신.

**C세션 다음 시작 번호: 10 (c-r3-10-...)**

## 작업 규칙

- 결과를 주제별로 파일 분할 저장한다.
- 하나의 질문/주제를 끝낼 때마다 즉시 파일에 저장한다. 머릿속에만 두지 마라.
- 컨텍스트 80% 차면 /compact 실행하고, compact 후 인덱스만 읽고 이어서 진행해라.
- 질문하지 말고 판단해서 끝까지 진행해라.
- 주제 끝날 때마다 `{세션}-index.md`에 한 줄 요약 추가.
- compact 후 → `{세션}-index.md` 한 파일만 읽고 재개.
