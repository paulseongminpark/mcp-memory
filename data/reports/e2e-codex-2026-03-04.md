## P1 — Critical Issues

### P1-1: init_db()가 v1 스키마만 생성 — fresh DB에서 v2 기능 즉시 실패
- **File**: `storage/sqlite_store.py:23`
- `init_db()`가 v1 `nodes`/`edges` 레이아웃만 생성. fresh database에서 `enriched_at`, `layer`, `quality_score`, `direction`, `base_strength` 등 v2 컬럼 누락.
- server 시작 시 `init_db()`만 호출하고 `migrate_v2.py` 자동 실행 안 함.
- **Impact**: `promote_node`, `daily_enrich.py` 등이 `sqlite3.OperationalError`로 실패.

### P1-2: E7(embedding_text) 완료 상태 미저장 — 반복 재처리
- **File**: `scripts/enrich/node_enricher.py:299-301`
- `enrich_node()`는 `updates`가 non-empty일 때만 `enrichment_status` 기록.
- E7의 `_apply` 브랜치는 ChromaDB만 업데이트하고 DB 필드 업데이트 없음.
- **Impact**: E7 타임스탬프 미저장 → 같은 노드 반복 처리 → 토큰 예산 낭비.

### P1-3: remember() 타입 검증이 v2 승격 타입 미포함
- **File**: `tools/remember.py:18-21`
- `remember()`가 `ontology/schema.yaml` 기준으로 타입 검증하는데, v2 승격 타입(`Observation`, `Signal`)이 schema에 없음.
- `VALID_PROMOTIONS`, `analyze_signals`, `get_becoming`은 이 타입 의존.
- **Impact**: Signal/Becoming 파이프라인 진입점 차단.

## P2 — Major Issues

### P2-1: 헤비안 업데이트 SQLite 연결 미닫힘 — 연결 누수
- **File**: `storage/hybrid.py:31-38`
- `_hebbian_update()`가 새 SQLite 연결을 열고 commit하지만 close() 안 함.
- `recall()` 결과 처리 시마다 실행 → 반복 검색으로 미닫힌 연결 누적.
- **Impact**: 장기 실행 시 lock/handle 압력 증가.

### P2-2: E19 타겟 ID 미검증 — 존재하지 않는 노드에 edge 삽입 시도
- **File**: `scripts/enrich/graph_analyzer.py:435`
- `e19_missing_links()`가 모델 출력의 `target_id`를 검증 없이 사용.
- Hallucinated ID로 foreign-key 에러 발생 가능 → 예외 핸들러가 나머지 제안도 드롭.
- **Impact**: 고아 노드 연결 시도의 부분 실패.

## P3 — Minor Issues

### P3-1: recall()에서 사용하지 않는 전체 그래프 빌드
- **File**: `tools/recall.py:21-23`
- `recall()`이 `get_all_edges()` + `build_graph()` 호출하지만 결과를 사용 안 함.
- **Impact**: 모든 recall 요청에 불필요한 O(E) 오버헤드. 그래프 커질수록 느려짐.

## Recommendations

1. `init_db()`에 v2 컬럼 포함하거나, server 시작 시 `migrate_v2.py` 자동 실행
2. E7 `_apply` 브랜치에서 `enrichment_status` 업데이트 추가
3. `schema.yaml`에 `Observation`, `Signal` 타입 추가 (v2 Becoming 파이프라인 전제)
4. `_hebbian_update()`에 `conn.close()` 추가 (finally 블록)
5. E19에 `target_id` 존재 여부 사전 검증 추가
6. `recall()`에서 미사용 `get_all_edges()`/`build_graph()` 제거
